Gaya hidup modern seringkali membuat seseorang sulit mengatur pola makan dan asupan kalori harian secara konsisten. Banyak masyarakat, khususnya mahasiswa dan pekerja, ingin menjaga berat badan dan kesehatan, namun tidak memiliki waktu atau alat praktis untuk mencatat dan memantau kalori harian mereka.
Aplikasi manajemen kalori harian membantu pengguna mencatat asupan kalori secara harian.

Tujuan Aplikasi : 
1. Mencatat makanan yang dikonsumsi.
2. Menghitung total kalori harian.
3. Memantau Berat Badan

Ruang Lingkup : 
1. CRUD Makanan Harian
2. Tampilan Kalori Total
3. Waktu Real-Time
4. Antarmuka Multibahasa
5. Login dan Registrasi
6. Ekspor/Impor Data
7. Penyimpanan Data
8. Enkripsi Password

Kebutuhan Teknis : 
1. Java (NetBeans IDE)
2. JDBC (MySQL/SQLite)
3. Threading
4. Generic Class
5. Serializable
6. SHA-256 untuk password
7. ResourceBundle dan file .properties

Stakeholder : 
1. Pengguna Umum
2. Mahasiswa (Developer)
